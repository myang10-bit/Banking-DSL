# Banking DSL

A simple banking system built in Python that models common banking operations such as opening accounts, deposits, withdrawals, transfers, and transaction history.

## Features
- Create and manage bank accounts
- Deposit and withdraw funds
- Transfer money between accounts
- Track transaction history with timestamps
- Input validation and exception handling

## Technologies
- Python
- Object Oriented Programming
- Unittest

## Purpose
This project was built to practice object oriented design, transaction handling, and writing clean, testable Python code.
